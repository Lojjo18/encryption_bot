This bot is a project created by a team of CS students from Princess Nora University.

This bot encrypts your messages in an efficient way!

WARNING! It's highly recomended to write the commands on a private room or through DM of the bot, also, you're responsible for remebering your keys after encryption.

List of commands (using 'ciph_' as a prefix):
encrypt: Encrypting the input text and sending the cipher text through DM (requires keys, 10 characters max).
decrypt: Decrypting the input cipher text and sending the cipher text through DM (requires keys, 10 characters max).
